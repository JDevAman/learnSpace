# SQL

## PSQL

| MySQL                 | PostGreSQL      |
| --------------------- | --------------- |
| SHOW DATABASES        | \l              |
| USE <db_name>         | \c <db_name>    |
| SHOW TABLES           | \dt             |
| DESCRIBE <table_name> | \d <table_name> |
| exit' or quit;        | \q              |

- In PostgreSQL, if table name is capital: you need enclose in double quotes.
